Time’s Up

The Game within the Game

How to Create a Game:

  1. Select “Create a Game” button.
  2. In the Game Info Section, name your game and select the game type.
  3. Choose your club.
  4. Choose the Winning Number Range you want to use for your game.
  5. Select whether you want “Auto Assigned” or “Player Assigned” to assign or choose your number.
  6. Select “Player Multiplier” if playing a game with bonuses – Select 1-3 multiplier.
  7. Choose whether you want to play with bonuses (explained in number 8 below).
  8. Remember (Copy and Paste) your Join Code Number to invite players to play.
  9. Click Create.
  10. Players then search the game code in the Lobby using the game code number, select the game, and hit join. The creator of the game must also select the game they just created and select join to join the game.
  11. If you selected “Player Assigned” feature, players choose their number and select “Join Game.”

Ways to Play for Swimming:

  • Choose which swimmers you will be playing the game with (e.g., swimmers on the same swim team or only the kids of parents playing the game).
  • Players score when a swimmer from their chosen group finishes the race with the same last number (1/100th of a second) as the player’s number.
  • When a player scores, click the Time’s Up Icon, select the multiplier, and hold down the button to claim your win.
  • If bonuses are chosen, notify each player at the beginning of the game how bonuses will be applied.

Ways to Play for Hockey:

  • Choose whether to use the last two digits of the time clock or the last digit.
  • When the game clock stops, the number it stops on determines the winner of the point.
  • When a player scores, click the Time’s Up Icon, select the multiplier, and hold down the button to claim your win.
  • If bonuses are chosen, notify each player at the beginning of the game how bonuses will be applied.

Common Questions:

  • What happens if someone claims a point they shouldn’t have or did not get a point when they should have?
    • The creator of the game can go into the game and adjust scores as needed.
  • How do you remove a player from a game?
    • The creator can go into the game’s settings and remove players.
  • How to report someone for objectionable content or inappropriate behavior?
    • Long press the comment or player name and follow the on-screen instructions to report.
